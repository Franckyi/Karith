# Installation

```groovy
implementation 'dev.franckyi:karith:0.1.0'
```

## Gradle (Kotlin)

```kotlin
implementation("dev.franckyi:karith:0.1.0")
```

## Maven

```xml
<dependency>
    <groupId>dev.franckyi</groupId>
    <artifactId>karith</artifactId>
    <version>0.1.0</version>
</dependency>
```
